## Assignments

### Basic Calculus & Solvers (HW1-HW3)
- **[HW1: Calculus Principles](file:///c:/Users/tanar/Documents/GitHub/_cm/hw1.py)**: Implementation of derivatives and integrals from first principles (definition of limits).
- **[HW2: Quadratic Solver](file:///c:/Users/tanar/Documents/GitHub/_cm/hw2.py)**: Solving $ax^2 + bx + c = 0$ with support for complex roots.
- **[HW3: Cubic Solver](file:///c:/Users/tanar/Documents/GitHub/_cm/hw3.py)**: Implementation of Cardano's method for cubic equations.
- **Method**: Original implementation with assistance for structural refinement.

### Numerical Methods (HW4)
- **[Homework 4: Polynomial Roots](file:///c:/Users/tanar/Documents/GitHub/_cm/Homework4/)**: Finding roots of high-degree polynomials using the Companion Matrix method and the Weierstrass (Durand-Kerner) method.
- **Method**: Original implementation of numerical algorithms with assistance for optimization.

### Geometry & Transformations (HW5)
- **[Homework 5: Geometry Construction](file:///c:/Users/tanar/Documents/GitHub/_cm/Homework5/)**: A library for geometric shapes (Point, Line, Circle, Triangle) including intersection detection and transformation matrices.
- **Method**: Original implementation using standard geometric algorithms.

### Machine Learning Foundations (MidTerm)
- **[MidTerm: Neural Network from Scratch](file:///c:/Users/tanar/Documents/GitHub/_cm/MidTerm/)**: A 2-layer feedforward neural network implementing backpropagation and the chain rule without high-level ML libraries.
- **Method**: Original mathematical implementation with assistance for documentation and training logic.

### Statistics & Information Theory (Week 8-9)
- **[Week 8: Statistical Hypothesis Testing](file:///c:/Users/tanar/Documents/GitHub/_cm/Week8/)**: Implementation of Z-tests and T-tests (One-sample, Independent, Paired) from scratch.
- **[Week 9: Information Theory](file:///c:/Users/tanar/Documents/GitHub/_cm/Week9/)**: Calculations for Entropy, KL Divergence, Mutual Information, and a (7,4) Hamming Code implementation.
- **Method**: Original implementation of statistical and information-theoretic formulas.

### Advanced Matrix Methods (Week 10)
- **[Week 10: Linear Algebra](file:///c:/Users/tanar/Documents/GitHub/_cm/Week%2010/)**: Implementations of LU Decomposition, Singular Value Decomposition (SVD), and Principal Component Analysis (PCA).
- **Method**: Original implementation using NumPy for base matrix operations.

### Signal Processing (Week 11)
- **[Week 11: Fourier Transform](file:///c:/Users/tanar/Documents/GitHub/_cm/Week11/)**: Manual implementation of the Discrete Fourier Transform (DFT) and its inverse (IDFT) without using FFT packages.
- **Method**: Original implementation of the Fourier summation.

### Differential Equations (HW13)
- **[Homework 13: ODE Solver](file:///c:/Users/tanar/Documents/GitHub/_cm/Homework13/)**: A general solver for homogeneous linear ordinary differential equations with constant coefficients.
- **Method**: Original implementation using characteristic equation root-finding.
